## 选择CPU

### 厂商

```
目前主流的桌面级CPU厂商有Intel英特尔和AMD两家，整体技术实力上面，Intel更强一些，这是因为在技术储备和产品力上，英特尔酷睿系列处理器为Intel打下了坚实的基础，而AMD近年来凭借锐龙系列处理器的出色表现，在桌面级处理器市场上正在迎头赶上。
说道两家的恩怨，可能就得从上个世界说起来的，但是对于小白玩家来讲，在日常使用上面，两家的处理器还是没有什么太大的区别的，一样可以打游戏，看电影，相对而言，AMD的处理器性价比更高一些。
目前的计算机架构依旧采用是冯诺依曼提出的结构计算器，他提出计算机硬件系统由运算器、存储器、控制器、输入设备、输出设备五大部件组成并规定了它们的基本功能。在冯诺依曼提出的计算机架构中，运算器是整个机器的中心，不但承担了各种运算任务，还是作为各个硬件的链接的桥梁，随着技术的发展，计算机的运算器和控制器合二为一，成为了现在的CPU （中央处理器），所以CPU的重要性不言而喻。
```

### 参数

```
CPU的运算运算速度直接决定了整机的性能表现，所以在选择CPU的时候一定要注意：
1）CPU的主频就是CPU的工作频率，理论上来讲主频越高，性能也就越强，CPU的主频=外频X倍频，单位为GHz，除了主频，另一个就是超线程技术了，超线程就是CPU可以同时执行多重线程，就能够让CPU发挥更大效率，那就是超线程(Hyper-Threading)技术，超线程技术减少了系统资源的浪费，可以把一颗CPU模拟成两颗CPU使用，在同时间内更有效地利用资源来提高性能，所以在选择的时候一定要看清你的处理器是否具有这样技术。
2）缓存同样是是一项关键的参数，目前处理器比较流行的是3阶缓存，
    L1　Cache(一级缓存)是CPU第一层高速缓存，分为数据缓存和指令缓存内置的L1高速缓存的容量和结构对CPU的性能影响较大，不过高速缓冲存储器均由静态RAM组成，结构较复杂，在CPU管芯面积不能太大的情况下，L1级高速缓存的容量不可能做得太大。一般服务器CPU的L1缓存的容量通常在32－256KB。
    L2　Cache(二级缓存)是CPU的第二层高速缓存，分内部和外部两种芯片。内部的芯片二级缓存运行速度与主频相同，而外部的二级缓存则只有主频的一半。L2高速缓存容量也会影响CPU的性能，原则是越大越好，以前家庭用CPU容量最大的是512KB，现在笔记本电脑中也可以达到2M，而服务器和工作站上用CPU的L2高速缓存更高，可以达到8M以上。
    L3　Cache(三级缓存)，分为两种，早期的是外置，现在的都是内置的。而它的实际作用即是，L3缓存的应用可以进一步降低内存延迟，同时提升大数据量计算时处理器的性能。降低内存延迟和提升大数据量计算能力对游戏都很有帮助。而在服务器领域增加L3缓存在性能方面仍然有显著的提升。比方具有较大L3缓存的配置利用物理内存会更有效，故它比较慢的磁盘I/O子系统可以处理更多的数据请求。具有较大L3缓存的处理器提供更有效的文件系统缓存行为及较短消息和处理器队列长度。
理论上来讲，缓存的容量越大，性能也就越高，但是受限于CPU的体积和制成的原因，现在一二级缓存无法做到更大，而三级缓存可以做的8M以上，所以看三级缓存的容量也是一个重要的参数，选大的一般不会出错。
3）一般来讲，目前绝大部分的网游对于处理器的单核性能要求较高，所以主频越高，在游戏表现上会更好，而一些专业的软件可能对于CPU的多核性能要求更高，在这里就需要多核性能较强的处理器来完成工作。
4）说了那么多，那么玩家究竟应该如何选择呢？首先要根据自己的预算，一般来讲，目前主流的游戏主机选择酷睿i5或者Ryzen 2600这个级别的处理器就能支持绝大部分的游戏的流畅运行。
另外在选择CPU的时候，玩家需要注意的是，一定要秉着买新不买旧的原则，尤其是小白用户，因为酷睿i7，i5系列处理器已经出了很多代，比如一代和目前的九代酷睿i5处理器性能相差就非常的多，所以用户在选择的时候一定要注意处理器的具体型号和发布年限，不能被一些无良的商家欺骗，目前最新的酷睿系列处理器为九代酷睿处理器，而AMD这边最新的处理器为AMD Ryzen锐龙2代处理器。
```


## 选择主板

### 参数

```
1.买主板最重要的是选对芯片组，一定要根据CPU来搭配合适的芯片组
2.大板小板性能几乎没有差距，有扩展需求的买大板、没有扩展需求的小板更加便宜
3.不管是大板还是小板，都建议优先选择有4根内存插槽的主板
4.喜爱RGB的玩家，建议选带有5V 3针的可编程RGB针脚的主板
```
