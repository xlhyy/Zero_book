# gitbook


- mac安装gitbook
```
安装node.js:
下载地址: https://nodejs.org/en/

查看是否成功:
node -V

安装gitbook:
sudo npm install gitbook-cli -g

查看是否成功:
gitbook -V

在文件夹内:
gitbook install
gitbook build

本地运行gitbook服务:
gitbook serve
```

- linux安装gitbook
```
安装node.js:
sudo apt-get install nodejs

查看是否成功:
node -V

安装npm:
sudo apt-get install npm

安装gitbook:
sudo npm install -g gitbook-cli

查看是否成功：
gitbook -V

在文件夹内:
gitbook install
gitbook build

本地运行gitbook服务:
gitbook serve
```
