# gitbook


- mac安装gitbook

```
安装node.js:
下载地址: https://nodejs.org/en/

查看是否成功:
node -V

安装gitbook:
sudo npm install gitbook-cli -g

查看是否成功:
gitbook -V

在文件夹内:
gitbook install
gitbook build

本地运行gitbook服务:
gitbook serve
```

```
新建gitbook项目流程：

github创建一个带有README.md的项目；
本地创建文件夹然后拉代码；
gitbook build;
创建SUMMARY.md文件;
gitbook serve运行项目;
```

- linux安装gitbook

```
安装node.js:
sudo apt-get install nodejs

查看是否成功:
node -v

安装npm:
sudo apt-get install npm

安装gitbook:
sudo npm install -g gitbook-cli

查看是否成功：
gitbook -v

在文件夹内:
gitbook install
gitbook build

本地运行gitbook服务:
gitbook serve
```
